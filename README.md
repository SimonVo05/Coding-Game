# 🚀 Catstronaut: Journey to Mars

A space themed coding adventure built with C# and Blazor. Players guide a cat astronaut through a short movement game, solve debugging challenges in an interactive terminal, and complete the mission to reach Mars.

---

## 📦 Stack

- C#
- ASP.NET Core Blazor
- Razor Components
- JavaScript
- CSS

---

## ✨ Features

- Story driven intro screen
- Cat movement mini game with keyboard controls
- Interactive coding terminal with debugging challenges
- Stage based progression system
- Code checking with challenge validation
- Mars landing win sequence
- Custom visuals and animations

---

## 🎮 How to play
- Start the adventure from the title screen
- Move the cat through the platform section
- Reach the next area to begin the coding challenges
- Fix the broken code in each stage
- Complete all challenges to finish the mission
- Land on Mars and win the game

---

## ⌨️ Controls
- Left Arrow or A to move left
- Right Arrow or D to move right
- Space to jump

---

## 🚦 Running the Project
- Clone the repository  
- Install dependencies
- Run the project 
- Open the local host link shown in the terminal in your browser
